## Getting Started

1. create `.env.development` file in root folder with values of:

```javascript
SANITY_STUDIO_PROJECT_URL = YOU_URL; // e.g http://localhost:3000
SANITY_STUDIO_PREVIEW_SECRET = YOUR_SECRET; // unique string
```

2. Run the development server:

```bash
npm run dev
# or
yarn dev
```
